@workspace utilise :
- .github/copilot/agent-ba-front.md
- .github/copilot/agent-unit-test-front.md

Analyse le composant ou service Angular sélectionné et génère les tests unitaires complets.

Respecter :
- les conventions Angular Testing existantes,
- les patterns frontend existants,
- les pratiques Craft du projet.

Toujours couvrir :
- cas nominaux,
- cas d’erreur,
- edge cases,
- boundary conditions,
- validations frontend,
- workflows UI,
- comportements async,
- erreurs API.

Générer :
- tests composants,
- tests services,
- tests formulaires,
- tests validators,
- mocks pertinents,
- assertions explicites,
- tests maintenables.

Ne jamais :
- générer des tests redondants,
- générer des mocks inutiles,
- générer des assertions faibles,
- générer des tests fragiles.

Utilise uniquement des caractères UTF-8 standards.
Ne jamais générer :
- d’emojis,
- de caractères Unicode invisibles,
- de caractères de contrôle,
- de caractères cachés,
- de texte corrompu.

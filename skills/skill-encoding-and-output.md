# ENCODAGE ET SORTIES

Toujours utiliser :
- UTF-8 standard.

Ne jamais générer :
- emojis,
- caractères Unicode invisibles,
- caractères de contrôle,
- caractères cachés,
- texte corrompu.

---

# SORTIES

Toujours produire :
- du texte lisible,
- du code maintenable,
- des réponses structurées,
- des formats cohérents.

---

# STYLE

Toujours :
- être précis,
- être factuel,
- éviter les réponses vagues,
- éviter les généralisations inutiles.

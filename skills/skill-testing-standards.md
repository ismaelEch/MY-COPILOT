# TESTING STANDARDS

Toujours générer :
- tests lisibles,
- tests maintenables,
- tests isolés,
- tests rapides.

---

# STRUCTURE AAA

## Arrange
Préparer le contexte.

## Act
Exécuter le comportement.

## Assert
Vérifier le résultat.

Toujours couvrir :

- edge cases,
- boundary conditions,
- validations,
- exceptions.

# MOCKS

Toujours :

- éviter les mocks inutiles.
